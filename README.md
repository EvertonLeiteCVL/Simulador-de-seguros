# Simulador de Seguros

Um simulador de seguros básico, desenvolvido em ASP.NET. Permite ao usuário calcular o valor de diferentes tipos de seguros.

## Descrição

Este projeto é um aplicativo web que simula o cálculo de seguros com base em informações fornecidas pelo usuário, como nome, data de nascimento, CPF e tipo de seguro. O aplicativo calcula o valor do seguro com base em uma tabela predefinida de preços.

## Funcionalidades

- Cadastro de informações do usuário
- Seleção do tipo de seguro
- Cálculo do valor do seguro com base nas informações fornecidas
- Validação de campos obrigatórios

## Tecnologias Utilizadas

- **ASP.NET**
- **C#**
- **HTML**
- **CSS**

## Como Executar o Projeto

1. Clone este repositório para sua máquina local.
2. Abra o projeto no Visual Studio.
3. Compile e execute o aplicativo.
4. Acesse a aplicação no seu navegador.
